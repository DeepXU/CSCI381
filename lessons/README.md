# Lessons Overview

This document provides a **central index of all lessons covered thus far** in the NCAE CyberGames training program.  
Each week corresponds to a directory within this `lessons/` folder, named `Week#`, where `#` represents the week number.

---

## Weekly Lesson Breakdown

| Week # | Topic Of The Week | Related Roles | Related Material |
|------|------------------|---------------|------------------|
| 0 | Pre-Competition Setup & Platform Access | CTF Master, Incident Responder | `docs/`, `issues/` |
| 1 | Base Configuration & Host Networking | Web Administrator, Network Administrator, Incident Responder | `docs/`, `scripts/` |
| 2 | Router Provisioning & Network Control | Network Administrator, SSH Administrator, Incident Responder | `docs/`, `scripts/` |
| 3 | Core Services Deployment (DNS, Web) | DNS Administrator, Web Administrator, Network Administrator, Incident Responder | `docs/`, `scripts/` |
| 4 | Remote Access & Secure Service Exposure (SSH, HTTPS, DNS) | SSH Administrator, Network Administrator, Incident Responder | `docs/`, `scripts/`, `issues/` |
| 5 | Database & File Services (PostgreSQL, SMB) | Database Administrator, File Server Administrator, Incident Responder | `docs/`, `scripts/`, `issues/` |

---

## Directory Structure Reference

- Each lesson lives in a corresponding folder:  
  `Week0/`, `Week1/`, `Week2/`, etc.
- Supporting documentation may exist in:
  - `docs/` – configuration and reference material
  - `scripts/` – automation and recovery tooling
  - `practiceLabs/` – supplemental exercises
  - `issues/` – known problems and resolutions

---

> **Note:** Some roles may not map directly to a single scoring category but are included where the lesson content directly strengthens their operational effectiveness during competition.
